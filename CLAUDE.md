# FiBace — Управление Арендной Недвижимостью

## Описание проекта
Платформа для арендодателей и арендаторов:
- Управление объектами недвижимости
- Бронирование и календарь
- Онлайн-оплата и подписки
- Документооборот
- Чат между сторонами

## Технологический стек
- Frontend: Next.js 14, TypeScript, Tailwind CSS
- Backend: Node.js + NestJS, PostgreSQL, Prisma ORM
- Mobile: React Native (iOS/Android)
- Payments: Stripe (подписки + разовые платежи)
- Security: JWT, шифрование, GDPR compliance

## Ключевые модули
1. Properties — каталог объектов
2. Bookings — бронирование и календарь
3. Payments — оплата и подписки
4. Documents — договоры, акты
5. Chat — коммуникация
6. Analytics — отчеты и статистика
